- to compile a c file
```bash
make filename
```
- to run a compiled file
```bash
./filename
```
- change directory
```bash
cd
```
- go to parent directory
```bash
cd ..
```
- remove file
```bash
rm
```
- create a folder / directory
```bash
mkdir
```
- remove a folder / directory
```bash
rmdir
```
- move a file
```bash
mv
```
- create a new file to code and open in vscode
```bash
code test.c
```
